Шрифт: Geist Sans. Скругления: 12–16px. Кнопки — pill / rounded-xl.

## Экраны MVP

### 1. Landing (`/`)
- Hero: «QuizLive — квизы в реальном времени»
- CTA: «Создать квиз» (организатор), «Присоединиться» (код комнаты)
- Ссылки: Вход / Регистрация

### 2. Auth (`/login`, `/register`)
- Email, пароль, имя
- Переключатель роли: Организатор / Участник

### 3. Dashboard (`/dashboard`) — организатор
- Сетка карточек квизов
- Кнопка «+ Новый квиз»
- Действия: Редактировать, Запустить

### 4. Quiz Editor (`/quiz/[id]/edit`)
- Мета: название, категория, таймер по умолчанию, режим баллов
- Список вопросов с drag-order (MVP: кнопки вверх/вниз)
- Модал/форма вопроса: тип, медиа, варианты, баллы

### 5. Session Lobby — host (`/host/[code]`)
- Крупный код комнаты
- Список подключившихся участников
- Кнопка «Начать квиз»

### 6. Live Host (`/host/[code]` — active)
- Текущий вопрос, таймер
- «Показать ответы» / «Следующий вопрос»

### 7. Join (`/join`)
- Поле 6-символьного кода

### 8. Live Player (`/play/[code]`)
- Waiting → Question (варианты) → Reveal (баллы) → Leaderboard

### 9. Profile (`/profile`)
- Вкладки: «Мои квизы» (орг.) / «Участие» (участ.)
